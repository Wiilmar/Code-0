[ HTML Concept Questions ]

[-] Que significa HTML?
HyperText Markup Language (Lenguaje de marcas de Hipertexto)

[-] Para que sirve HTML?
Permite crear la estructuracion de un sitio web

[-] Que es el Hypertexto?
El hipertexto es un sistema organizado que permite vincular diferentes documentos o secciones dentro de un documento mediante enlaces, facilitando una lectura no lineal.

[-] Para que sirve <!DOCTYPE html>?
Doctype es una declaración que indica al navegador el tipo de documento que se va a renderizar. En este caso, al ser HTML, utilizará los estándares más modernos, permitiendo la compatibilidad en diferentes navegadores.

[-] Para que sirve la etiqueta: <Head>?
La etiqueta <head> proporciona información sobre el contenido y la estructuración del documento a los motores de búsqueda, esta información no es visible al público. También permite vincular otros recursos como, por ejemplo, el estilo y la interactividad.
