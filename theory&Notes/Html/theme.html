Etiquetas HTML5 para enfasis de texto

  1. Negrita
    • `<b>`: Negrita sin énfasis especial.
    • `<strong>`: Negrita con significado fuerte o urgente.

      • Este texto es importante.

      Este texto es muy importante.

  2. Cursiva
    • `<i>`: Cursiva sin énfasis especial.
    • `<em>`: Cursiva con énfasis.

      • Este texto es en cursiva.

      Este texto es en cursiva y con énfasis.

  3. Subrayado:
    • `<u>`: Texto subrayado

      • Este texto está subrayado.

  4. Tachado:
    • `<s>`: Texto ya no relevante o incorrecto.
    • `<del>`: Texto eliminado del documento.

      • Este texto está tachado.

      Este texto está eliminado.

  5. Subindice:
    • `<sub>`: Texto como subíndice.

      • El agua es H2O.

  6. Superindice:
    • `<sup>`: Texto como superíndice.

      • El área de un cuadrado es a2.
