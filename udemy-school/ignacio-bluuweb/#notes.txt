[o] Que son las convenciones de nomenclatura?
R/: Son un conjunto de reglas y prácticas recomendadas para nombrar elementos.

[o] Cuales son las reglas de nomenclatura que aplicare?
R/: 
- Regla para nombre de variables y funciones: camelCase
- Regla para nombre de constantes: UPPER_SNAKE_CASE
- Regla para nombre de clases: PascalCase
- Regla para nombre de carpetas y archivos: kebab-case
(No escribir simbolos, espacios y tildes, hazlo simple)

[o] Para que sirve la etiqueta <p>?
R/: Define un párrafo de texto.

[o] Para que sirve la etiqueta <title>?
R/: define el título del documento web, visible en la pestaña del navegador.

[o] Para que sirve el atributo title?
R/: proporciona información adicional sobre un elemento específico, visible como un tooltip.
